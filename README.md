## What Each Assignment Covers

- `hw1`: Implements core image processing operations in C++, including RGB-to-grayscale conversion, 4-level grayscale quantization, 3x3 box blur convolution with padding, and stride-2 image downsampling with result comparison.
- `hw2`: Implements grayscale image denoising with median filtering, Gaussian filtering, and combined median-Gaussian pipelines, together with PSNR-based parameter search, histogram rendering, and report/result generation.
- `hw3`: Implements grayscale preprocessing, Gaussian smoothing, custom Canny edge detection, Hough-line voting, and quadrilateral-based card border extraction for three test images, together with report and result generation.
